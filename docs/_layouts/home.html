Plan. Test. Report. Done.

Hailstorm takes the pain out of distributed JMeter tests by automating the whole process from coordinating the tests to report generation.

Introduction

Hailstorm is a load generation application capable of generating massive amounts of load on a test system. Hailstorm uses Apache JMeter to generate the load on the system under test. The application provides a command-line interface (CLI) and web based user interface to configure the test environment, start tests, stop tests and generate reports.

Hailstorm can generate load using AWS EC2 instances or use physical/virtual machines in a data center. It generates editable reports (.docx files). This is a sample PDF . The editable file is available on request . All the contents of those files is autogenerated. There are highlighted sections that suggest adding more information to explain a chart or table.

For more information, check out the About page.
